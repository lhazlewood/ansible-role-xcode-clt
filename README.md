# Ansible Role: Xcode Command Line Tools

Ansible role that installs Xcode Command Line Tools on Mac OS X >= 10.10 (Yosemite, El Capitan, Sierra, High Sierra, etc).

## Example Playbook

```yaml
- hosts: localhost
  roles:
    - lhazlewood.xcode-clt
```
